# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview
This project focuses on building a Machine Learning model to predict the likelihood of heart disease in a patient based on medical attributes. 
The goal is to assist early diagnosis and support medical decision-making using data-driven techniques.
---

## 🎯 Problem Statement
Heart disease is one of the leading causes of death globally. Early prediction can help in prevention and timely treatment. This project aims to 
classify whether a patient is at risk of heart disease based on input health parameters.
---

## 📊 Dataset
- Source:  Heart Disease Dataset ( Kaggle dataset)
- Features include:
  - Age
  - Sex
  - Chest pain type
  - Blood pressure
  - Cholesterol level
  - Maximum heart rate
  - Exercise-induced angina
  - And other clinical features

---

## 🤖 Machine Learning Models Used

## 📈 Model Evaluation
The models were evaluated using:
- Accuracy Score
- Confusion Matrix

---

## 📊 Results
